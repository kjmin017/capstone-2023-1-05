## 5. 사용법
사용법<br> 
1. text칸에 질문하고자 하는 내용을 입력한다.
2. text칸 밑의 Clear칸을 클릭하면 내용을 지울 수 있다.
3. text칸 밑의 Submit칸을 클릭하면 질문이 제출된다.
4. 질문에 대한 답변이 Output칸에 입력된다.
